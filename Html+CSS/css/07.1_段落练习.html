Angelababy简介

基本信息

Angelababy（杨颖），1989年2月28日出生于上海市，华语影视女演员、时尚模特。

2003年，Angelababy以模特身份出道，此后，她因担任时尚模特而在香港崭露头角。2007年，她开始将工作重心转向大银幕。2011年，她在爱情片《夏日乐悠悠》中首次担任女主角。2012年，凭借言情片《第一次》获得第13届华语电影传媒大奖最受瞩目女演员奖。

早年经历

1989年，Angelababy出生在上海，并在那里成长，因此她的童年时光都是在上海的弄堂里度过的 [11] 。Angelababy自小和外婆感情深厚，更因为父母一直为工作奔波，在小学一年级时便要独自放学回家。而她的小学则就读于上海市徐汇区长乐学校 [12] 。此外，杨颖还有一个小自己七岁的弟弟杨帆。13岁时，她前往香港并加入了Talent Bang。此后、她还担任了Viva Club Disney的主持。

个人生活

2011年2月，Angelababy正式确定恋爱关系。 2014年1月，AngelaBaby正式公开承认黄晓明是其结婚对象 。

2015年5月27日，Angelababy与黄晓明在青岛民政局领证登记； 10月8日，两人在上海展览中心举办婚礼。 2016年，Angelababy怀孕 。 2017年1月17日晨，Angelababy在香港诞下一子，小名为“小海绵”。
